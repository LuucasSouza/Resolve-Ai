# End-to-End Validation Protocol

## Steps

1. Select a concrete scenario.
2. Select entry mode.
3. Run builder intake.
4. Run Discovery.
5. Run Product.
6. Run Architecture.
7. Run Decision and ADRs.
8. Run Risk.
9. Run Optimization.
10. Run Execution.
11. Run Documentation and Handoff.
12. Run Reflection.

## Quality Gates

Every stage must produce files, decisions, risks, handoff and improvement items.
