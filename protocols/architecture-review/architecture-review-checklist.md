# Architecture Review Checklist

- [ ] Is the problem clear?
- [ ] Are options real and distinct?
- [ ] Are criteria appropriate?
- [ ] Are trade-offs explicit?
- [ ] Are risks visible?
- [ ] Is selected option aligned with product scope?
- [ ] Is reversibility classified?
- [ ] Does this create lock-in?
- [ ] Can the team operate it?
- [ ] Is there a simpler acceptable alternative?
