# Protocol System

## Objective

Define protocols as operational contracts for repeated AI-SEOS activities.

## Scope

Protocols cover intake, discovery, product, architecture, decision, risk, optimization, execution, documentation, handoff, reflection, release and community work.

## Principles

- Every protocol has input and output contracts.
- Every protocol has quality gates and failure modes.
- Every protocol can be executed by humans or AI agents.
- Protocols must not duplicate each other.

## Responsibilities

Protocol Maintainers own the registry, lifecycle, taxonomy and review process.

## Next Steps

Promote high-usage protocols from draft to trial after real project validation.
