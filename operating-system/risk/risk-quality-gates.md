# Risk Quality Gates

| Gate | Requirement |
|---|---|
| Identification Gate | Risk has future event/condition, cause and impact |
| Classification Gate | Category, source artifact and owner assigned |
| Assessment Gate | Probability, impact, level and rationale documented |
| Response Gate | Strategy, mitigation/acceptance and due date documented |
| Approval Gate | High/critical risks have approval or escalation |
| Monitoring Gate | Trigger signals and review cadence exist |
