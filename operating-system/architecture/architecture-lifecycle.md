# Architecture Lifecycle

1. Receive Product Handoff.
2. Validate Architecture Inputs.
3. Extract Architectural Drivers.
4. Define Quality Attribute Scenarios.
5. Model Domain.
6. Define System Boundaries.
7. Identify Architecture Options.
8. Compare Trade-offs.
9. Select Baseline Architecture.
10. Create Architecture Views.
11. Create ADRs.
12. Assess Architecture Risks.
13. Architecture Quality Gate.
14. Architecture Handoff.
