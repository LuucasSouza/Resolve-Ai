# Optimization Quality Gates

| Gate | Requirement |
|---|---|
| Baseline Gate | Current decision/design, target and baseline cost/complexity/risk understood |
| Dimension Gate | Relevant dimensions selected and irrelevant dimensions skipped explicitly |
| Option Gate | At least two options plus keep-current when meaningful |
| Trade-off Gate | Benefits, new risks and sacrifices are explicit |
| Artifact Gate | ADRs, architecture docs, scope and execution constraints updated |
