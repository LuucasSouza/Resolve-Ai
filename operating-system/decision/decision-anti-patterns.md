# Decision Anti-Patterns

- Preference disguised as architecture.
- Premature enterprise architecture.
- Tool-first decision.
- False binary choice.
- Hidden stakeholder constraint.
- No reversal strategy.
- Highest matrix score accepted blindly.
- Decision without monitoring or revalidation trigger.
