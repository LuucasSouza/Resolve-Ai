# Decision Quality Gates

| Gate | Requirement |
|---|---|
| Context Gate | Business, product, technical impact and constraints are known |
| Framing Gate | Decision question, scope, non-scope and class are explicit |
| Alternatives Gate | At least three meaningful alternatives or documented exception |
| Criteria Gate | Criteria and weights are explicit and justified |
| Trade-off Gate | Benefits, costs, sacrifices and future constraints are documented |
| Risk Gate | Risks are identified and high risks have mitigation or acceptance |
| Reversibility Gate | Reversibility, rollback path and migration cost are assessed |
| Documentation Gate | ADR or decision log exists and downstream agents can consume it |
