# Vibe Coder Template Pack

## Objetivo

Dar estrutura para usuarios que constroem com ferramentas de IA sem perder velocidade.

## Escopo

Inclui intake, escolha de stack, plano de sessao com IA, implementation slices, prompt para Codex, validacao, debugging, refactor e deploy readiness.

## Principios

- Construir em slices pequenos.
- Validar depois de cada slice.
- Evitar complexidade prematura.
- Manter o projeto explicavel.
