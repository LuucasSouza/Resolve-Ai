# Full Lifecycle Template Pack

## Objetivo

Conectar intake, discovery, product, architecture, decision, risk, optimization, execution, documentation, handoff e reflection em um fluxo reutilizavel.

## Escopo

Este pack aponta para 	emplates/lifecycle/ e serve como trilha padrao para projetos que precisam atravessar o AI-SEOS completo.

## Principios

- Um artefato minimo por etapa.
- Handoff explicito entre etapas.
- Traceability de problema a melhoria continua.
