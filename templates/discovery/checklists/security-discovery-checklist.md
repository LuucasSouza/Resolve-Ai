# Security Discovery Checklist

- [ ] Authentication requirements.
- [ ] Authorization model.
- [ ] Sensitive data types.
- [ ] PII handling.
- [ ] Payment data handling.
- [ ] Secrets management.
- [ ] Audit logging needs.
- [ ] Rate limiting needs.
- [ ] Abuse cases.
- [ ] Backup/restore needs.
- [ ] Access review process.
- [ ] Security handoff required?
