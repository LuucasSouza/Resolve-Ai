# ADR 0121 — Adopt Execution Prompt Generation

## Status

Accepted

## Contexto

Resolve Aí deve ajudar usuários a trabalhar com agentes sem perder controle.

## Decisão

Gerar prompts prontos para agentes de IA, mas não executar automaticamente.

## Consequências

Usuário pode copiar prompts seguros e controlados.

## Alternativas consideradas

- Executar prompts automaticamente.
- Não gerar prompts.

## Impacto no usuário

Reduz ambiguidade na hora de pedir implementação.

## Riscos

Prompts ainda exigem revisão do usuário.

## Relação com roadmap

Prepara guided execution sem automação destrutiva.
