# ADR-0011 - Adopt Discovery Depth Levels

## Status

Accepted

## Context

Not every initiative needs the same discovery depth. A low-risk internal change and an enterprise regulated product should not use identical process weight.

## Decision

Adopt D0, D1, D2 and D3 discovery depth levels.

## Alternatives Considered

- One discovery process for all initiatives.
- Fully ad hoc discovery.
- Progressive depth model.

## Consequences

Positive: discovery can scale with risk, uncertainty and cost.

Negative: agents must choose the correct depth and justify it.

## Reversibility

High. Depth levels can be renamed or expanded by future ADR.

## Follow-up Actions

- Add examples for each depth level.
- Validate depth selection rules in real discoveries.
