# ADR 0052 - Adopt Template Packs as First-Class Framework Assets

## Status

Accepted

## Version

0.1.0

## Date

2026-07-04

## Context

Sprint 6 turns AI-SEOS from narrative operating documentation into a reusable template system for humans and AI agents.

## Decision

Template packs are official framework assets, not optional examples.

## Consequences

Templates become governed assets with owners, quality gates, examples, registry entries and maintenance obligations.

## Responsibilities

Template Maintainers keep the related templates aligned with Entry Modes, lifecycle contracts, engines, ADRs and examples.

## Next Steps

Validate real usage in Sprint 7 through protocols, cases, anti-patterns and best practices.
