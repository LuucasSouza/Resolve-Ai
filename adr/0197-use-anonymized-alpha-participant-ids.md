# ADR 0197 - Use Anonymized Alpha Participant IDs

## Status

Accepted

## Context

Validacoes com usuarios podem capturar dados pessoais desnecessarios.

## Decision

Usar IDs anonimos como `NTB-01`, `VC-01` e `PE-01`.

## Consequences

- Reduz risco de privacidade.
- Facilita analise por perfil.
- Evita exposicao de participantes.
