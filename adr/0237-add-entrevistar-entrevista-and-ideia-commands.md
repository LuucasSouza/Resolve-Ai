# ADR 0237: Add entrevistar, entrevista, and ideia Commands

## Status

Accepted

## Date

2026-07-06

## Context

The public promise says: "Me dá o problema ou a ideia, e eu te ajudo a resolver."

## Decision

Add `resolve-ai entrevistar` with aliases `resolve-ai entrevista` and `resolve-ai ideia`.

## Consequences

Users can start from an idea using Brazilian Portuguese language. Existing commands remain compatible.

## Alternatives Considered

- Use only `diagnosticar`: too file-oriented.
- Use an English command: misaligned with Portuguese-first UX.

## Relation to Real Feedback

Feedback requested a discovery interview command and clearer guided flow.
