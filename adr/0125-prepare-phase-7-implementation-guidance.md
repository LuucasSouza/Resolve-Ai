# ADR 0125 — Prepare Phase 7 Implementation Guidance

## Status

Accepted

## Contexto

Depois do planejamento, o próximo passo natural é preparar execução de uma tarefa.

## Decisão

Phase 7 deve focar em orientar execução segura de uma tarefa, não em automação destrutiva.

## Consequências

O roadmap avança mantendo confiança e controle humano.

## Alternativas consideradas

- Implementar execução automática completa.
- Parar no planejamento.

## Impacto no usuário

Usuário terá ponte entre backlog e execução assistida.

## Riscos

Nome e escopo do comando futuro ainda precisam decisão.

## Relação com roadmap

Define Phase 7 — Guided Execution Prep.
