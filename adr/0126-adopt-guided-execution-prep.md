# ADR 0126 — Adopt Guided Execution Prep

## Status

Accepted

## Contexto

Após diagnosticar e planejar, o Resolve Aí precisa preparar uma tarefa executável sem executar mudanças.

## Decisão

Adicionar `resolve-ai preparar` para preparar execução sem alterar código.

## Consequências

Cria ponte segura entre backlog e implementação futura.

## Alternativas consideradas

- Executar automaticamente.
- Parar no planejamento.

## Impacto no usuário

Usuário recebe uma tarefa pequena com escopo, prompt, riscos e validação.

## Riscos

Preparação ainda exige aprovação humana.

## Relação com roadmap

Prepara Phase 8 de validação.
