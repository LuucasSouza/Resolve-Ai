# ADR 0123 — Adopt Status Integration for Planning State

## Status

Accepted

## Contexto

`status` deve refletir onde o projeto parou.

## Decisão

`resolve-ai status` deve mostrar último planejamento.

## Consequências

Retomada fica mais clara.

## Alternativas consideradas

- Deixar plano apenas em documentos.

## Impacto no usuário

Usuário vê próxima ação planejada no terminal.

## Riscos

Estado local precisa seguir schema compatível.

## Relação com roadmap

Apoia continuidade entre fases.
