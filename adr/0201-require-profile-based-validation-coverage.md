# ADR 0201 - Require Profile-Based Validation Coverage

## Status

Accepted

## Context

O Resolve Ai atende tres publicos diferentes.

## Decision

Validar separadamente Non-Technical Builder, Vibe Coder e Professional Engineer.

## Consequences

- Feedback fica mais preciso.
- Pode revelar tensoes entre simplicidade e rigor.
- Release precisa equilibrar publicos.
