# ADR 0133 — Extend Status with Prepared Task Summary

## Status

Accepted

## Contexto

O usuário precisa ver rapidamente a próxima tarefa preparada.

## Decisão

`resolve-ai status` mostra resumo da tarefa preparada.

## Consequências

Status passa a refletir diagnóstico, plano e preparo.

## Alternativas consideradas

- Mostrar apenas docs gerados.

## Impacto no usuário

Melhora retomada.

## Riscos

Status pode ficar longo.

## Relação com roadmap

Apoia continuidade entre comandos.
