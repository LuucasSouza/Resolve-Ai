# ADR 0117 — Keep Planning Read-Only for Product Code

## Status

Accepted

## Contexto

Planejamento deve preparar implementação, não alterar o produto.

## Decisão

`planejar` pode criar docs e estado do Resolve Aí, mas não altera código do projeto.

## Consequências

Reduz risco e mantém confiança no CLI.

## Alternativas consideradas

- Criar tarefas e já alterar código.
- Executar scripts do projeto.

## Impacto no usuário

O projeto analisado permanece seguro.

## Riscos

O usuário ainda precisa executar a implementação manualmente ou com agente.

## Relação com roadmap

Mantém Phase 6 como preparação.
