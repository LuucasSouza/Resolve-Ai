# ADR 0131 — Require Human Approval Before Implementation

## Status

Accepted

## Contexto

Execução de código ainda não tem modelo de aprovação.

## Decisão

Todo pacote de execução exige aprovação humana.

## Consequências

`approvalRequired` é sempre true.

## Alternativas consideradas

- Autoaprovação.
- Execução automática.

## Impacto no usuário

Mantém controle humano.

## Riscos

Fluxo é mais manual.

## Relação com roadmap

Base para execução segura futura.
