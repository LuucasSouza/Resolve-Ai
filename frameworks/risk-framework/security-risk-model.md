# Security Risk Model

## Categories

Authentication, authorization, data isolation, secrets management, input handling, webhook security, data protection, auditability, dependency security, infrastructure security, abuse/fraud and AI security.

## Automatically Critical

Cross-tenant data exposure, production secrets committed, public unauthenticated sensitive data, exposed payment credentials, admin privilege escalation, destructive unvalidated AI tool execution.
