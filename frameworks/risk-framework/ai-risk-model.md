# AI Risk Model

## Risks

Prompt injection, data leakage, unsafe tool execution, model drift, hallucination, nondeterminism, hidden token cost and agent overreach.

## Controls

Scope tools, validate outputs, minimize sensitive context, log safely, require human review for high-impact actions and monitor cost per workflow.
