# Decision Confidence Model

| Confidence | Meaning |
|---|---|
| Low | Many unknowns, weak evidence, high uncertainty |
| Medium | Reasonable evidence with material assumptions |
| High | Strong evidence, clear constraints, stable context |
| Conditional | Acceptable only while assumptions hold |

## Sensitivity Questions

- If delivery speed weight changes, does recommendation change?
- If security weight increases, does recommendation change?
- If scale increases 10x, does recommendation change?
- If budget decreases 50%, does recommendation change?
- If compliance becomes stricter, does recommendation change?
