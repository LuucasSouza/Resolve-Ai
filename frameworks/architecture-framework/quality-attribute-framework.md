# Quality Attribute Framework

## Scenario Format

```text
Given [context]
When [stimulus]
The system should [response]
Within [measure]
Under [constraints]
```

## Attributes

Availability, reliability, performance, scalability, security, privacy, maintainability, observability, operability, cost.
