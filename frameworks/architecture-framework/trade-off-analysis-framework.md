# Trade-off Analysis Framework

## Criteria

Product fit, complexity, maintainability, security, reliability, scalability, cost, team capability, operational burden, vendor lock-in, reversibility, time to market.

## Rule

Significant decisions should compare conservative, balanced and ambitious options.
