# Domain and Integration Modeling

## Objetivo

Entender domínio, boundaries, data ownership, data movement and external systems before technical choices.

## Domain Concept Types

Actor, Entity, Value Object, Aggregate, Domain Event, Policy, Process and External System.

## Integration Types

Synchronous API, Async Event, Webhook, Batch Import/Export, Embedded SDK and Human Operational.

## Handoff Includes

Domain concepts, candidate boundaries, data ownership, integration map, critical dependencies, failure modes and decisions requiring ADR.
